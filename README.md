# GEST-JV

## Run application

- download with git :

```
git clone git@github.com:barmic/GEST-JV.git
```

- build and run with maven :

```
mvn clean package spring-boot:run
```

## Usage

- ``http://localhost:8080/list`` see all games of database
- ``http://localhost:8080/insert`` add game in database
